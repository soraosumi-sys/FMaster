# FlashMaster ⚡️
A beautiful, glassmorphic English Flashcard App.

## How to Run
Simply open `index.html` in your web browser.
No installation or server required.

## Features
- **Smart Flashcards**: 
  - "OK" reduces the card's frequency.
  - "?" (Unknown) increases the card's frequency (8x boost) and adds it to the review list.
- **Unknown List**: Track your most difficult words and see how many times you missed them.
- **Add Words**: Easily add new vocabulary to your deck.
- **Data Persistence**: Your progress is saved automatically in your browser's local storage.

## Tech Stack
- Vanilla HTML5 / CSS3 / JavaScript
- Google Fonts (Inter, Noto Sans JP)
- Lucide Icons
